JavaScript Concepts Overview

1. Fetch()

It is used to make HTTP Requests like getting data from an API or sending data to a server.

Why we use fetch()

  • To get data from APIs
  • To send data to servers
  • To update or delete data on a server

Where we use fetch()

  • Web applications that display live or dynamic data.

2. Table

A table is a way to display data in rows and columns.

Tags:

Tag Description
<table>Creates a table
<tr>Table row
<th>Table header (bold)
<td>Table data cell

foreach()

  • Executes the function for each element
  • It’s often used to loop through arrays and perform actions

3. .then()

  • It is a function that runs one after another
  • Runs when promise is fulfilled

Array Methods

  • map() - Create a new array by applying a function; it can return a value
  • filter() - Return a new array that passes a condition; returns all elements that match the condition
  • reduce() - Reduce the array to a single value by using accumulator

4. DOM

  • It is a Document Object Model
  • It is an API method to handle HTML structures and values
  • It converts the HTML file into an object-like tree structure
  • It is the live representation of an HTML page
  • JavaScript can use it to read, modify, delete, and add elements dynamically
  • Methods include selectors, properties, events, and event listeners
  • The selector is used for interaction — to change, modify or manipulate elements
  • The property is used to show on the screen
  • The event is used to tell the browser what the user is doing
