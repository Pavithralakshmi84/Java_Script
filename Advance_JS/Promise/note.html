JavaScript Promises FAQ

Q: What is a Promise?

A Promise is like a "contract" between your code and a future result. It says — "I’ll do something, and once done, I’ll either give you a result (resolve) or an error (reject)."

Q: What is .then() used for?

.then() runs when the Promise is resolved — that means the success case. We can chain multiple .then() calls to run steps one after another in sequential flow.

Q: What is .catch() used for?

.catch() handles error cases. If any promise fails (rejects), then .catch() will run to handle the error.

Q: Why do we use return inside .then()?

Each .then() should return the next promise so the chain continues properly.
